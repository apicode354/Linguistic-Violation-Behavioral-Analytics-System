# Linguistic-Violation-Behavioral-Analytics-System
Nama Tool: "LangSentry — Linguistic Violation & Behavioral Analytics System"Masalah yang Dipecahkan: Pencatatan buku fisik pelanggaran (Spy Book) yang sering hilang, subjektif, ketidakmampuan melacak pelanggar kambuhan secara sistemis, serta ketiadaan standardisasi hukuman edukatif.Pengguna: Staf Komando Bagian Bahasa / Ustadz Penegak Disiplin.Input: Input ketukan cepat (tap-input) nama santri dan jenis pelanggaran bahasa daerah/Indonesia oleh Ustadz di lapangan.Output: Log pelanggaran real-time, grafik tren frekuensi pelanggaran mingguan, dan rekomendasi tipe hukuman edukatif otomatis berbasis data historis.Platform: Mobile-First Web App (PWA Optimized).Teknologi (Rencana): HTML5, Alpine.js, Tailwind CSS, Google Apps Script (Backend DB Sheets via REST API).Struktur Data & Algoritma: Relational Tables (Santri ID $\leftrightarrow$ Violation Code $\leftrightarrow$ Timestamp); Algoritma Klasifikasi Aturan (Rule-Based Classification) untuk eskalasi sanksi otomatis.
